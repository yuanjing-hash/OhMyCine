# OhMyCine — 产品架构总览

> THE NORTH STAR OF YOUR CINEMA

## 1. 产品愿景

OhMyCine 是一个**开源、全平台、自托管**的家庭影院生态系统。

- **播放器**拥有顶级画质与沉浸式体验（Cinema OS 风格 UI、液态玻璃设计语言）
- **后端**拥有自动化媒体流水线（发现→下载→转移→入库→通知，PT站点管理、追更、STRM生成、302直连播放）
- **插件市场**让社区驱动功能扩展
- **CLI工具**让高级用户和自动化脚本无缝集成

**核心特点**：全平台 + 开源 + 独立后端 + 自动化媒体管理 + 用户自带API Key

## 2. 产品矩阵

| 组件 | 名称 | 定位 | 技术栈 |
|------|------|------|--------|
| 播放器 | **OhMyCine Player** | 跨平台桌面/移动播放器 | Tauri v2 + Vue 3 + libmpv (嵌入式) |
| 后端 | **OhMyCine Server** | 媒体流水线 + 302代理 | Go + SQLite/PostgreSQL |
| 插件市场 | **OhMyCine Hub** | 社区插件分发平台 | VitePress (静态站点) |
| CLI | **omc** | 命令行管理工具 | Go (cobra) |

**核心设计理念**：
- **Player 独立可用**：不依赖 Server，原生连接 Emby/Jellyfin/Alist/CloudDrive2
- **Server 是增强层**：提供媒体流水线（发现→下载→转移→入库）、追更、302代理等高级功能
- **三层存储架构**：连接管理 → 存储目标 → 分类规则，清晰解耦
- **双向配置同步**：Player 连接 Server 时自动同步数据源配置

**数据流说明**：
- **Player → Emby/Jellyfin**：Player 原生调用 Emby/Jellyfin REST API，直接浏览和播放
- **Player → Alist/CloudDrive2**：通过 WebDAV 协议直接访问云盘文件
- **Player ↔ Server**：可选连接，同步配置、获取下载任务/追更状态
- **Server 媒体流水线**：发现页聚合PT搜索 → 下载器下载 → 自动分类 → 转移到存储目标 → 生成STRM(网盘) → 通知Emby刷新
- **Server → 云盘CDN**：302代理引擎返回重定向，客户端直连CDN

## 4. 核心设计原则

### 4.1 用户数据主权
- 所有数据存储在用户自己的服务器/NAS上
- AI功能使用用户自己的API Key（OpenAI/Claude/本地模型）
- 不收集任何用户数据，不依赖任何中心化服务

### 4.2 模块化 & 可插拔
- 每个组件可独立部署、独立使用
- **Player 独立可用**：原生支持 Emby/Jellyfin/Alist/CloudDrive2，无需 Server
- Server 可以单独部署（用 Web UI 管理，不需要 Player 客户端）
- 插件系统支持热加载

### 4.3 开源优先
- GPL-3.0 License
- 社区驱动开发
- 透明的开发路线图

### 4.4 性能至上
- 302直连播放：零带宽消耗，云盘CDN直接串流
- MPV引擎：硬件解码、128线程并行、零丢帧
- 预取缓存：行为预测预加载

## 5. 部署模式

### 模式A：全栈部署（推荐）
```
Docker Compose 一键部署：
- OhMyCine Server (Go)
- Emby/Jellyfin (媒体服务器)
- qBittorrent (下载器)
- Alist (可选，云盘聚合)
```

### 模式B：Player + Server 分离
```
Player 安装在桌面/笔记本
Server 部署在 NAS/服务器
通过 REST API 通信
```

### 模式C：Player Only
```
仅使用播放器功能
播放本地文件 / 手动添加URL
无需Server
```

### 模式D：Server Only + 第三方客户端
```
Server 部署在NAS
使用 Emby/Jellyfin 官方客户端
OhMyCine Server 提供302代理 + 媒体管理
```

## 6. 技术选型总览

| 层级 | 技术 | 理由 |
|------|------|------|
| Player UI Framework | Tauri v2 | 全平台(Windows/macOS/Linux/Android/iOS)、包体小、性能好 |
| Player UI Library | Vue 3 + TypeScript | 响应式、生态成熟、与Tauri集成好 |
| Player Styling | UnoCSS + CSS Variables | 原子化CSS + 液态玻璃设计系统 |
| Player Playback | libmpv (嵌入式) | 格式支持最广、HDR/DV、硬件解码、沉浸式渲染 |
| Player Animation | Motion Vue / GSAP | Cinema OS级流畅动画 |
| Server Language | Go | 并发模型好、交叉编译简单、Alist生态一致 |
| Server Database | SQLite (嵌入) + PostgreSQL (可选) | 轻量默认、可扩展 |
| Server API | REST + WebSocket | 标准化、易集成 |
| Server Deploy | Docker + 二进制 | 灵活部署 |
| CLI | Go + Cobra | 与Server共享代码 |
| Hub | Vue 3 + VitePress | 静态站点 + 插件注册表 |
| CI/CD | GitHub Actions | 自动构建、自动发布 |
| 文档 | VitePress | Vue生态一致 |
